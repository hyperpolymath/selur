= Contributing to selur
:toc:

== Welcome

Thank you for contributing to selur, the Ephapax-linear WASM sealant for Svalinn and Vörðr!

== Development Setup

[source,bash]
----
# Clone repository
git clone https://github.com/hyperpolymath/selur
cd selur

# Install dependencies
# - Zig (for WASM compilation)
# - Ephapax compiler (for .eph → IR)
# - Idris2 (for formal verification)
# - Rust (for bindings)

# Build
just build

# Test
just test

# Verify proofs
just verify
----

== Code Standards

* **Ephapax**: Follow linear type conventions, document regions
* **Zig**: Use `zig fmt`, optimize for WASM32 size
* **Idris2**: Literate proofs, clear theorem statements
* **Rust**: `cargo fmt` and `cargo clippy` clean
* **SPDX headers**: All files must have license headers

== Pull Request Process

1. Fork the repository
2. Create a branch: `git checkout -b improve-ipc-latency`
3. Make changes with SPDX headers
4. Run `just check` and `just test`
5. Run `just verify` if modifying Idris2 proofs
6. Commit with conventional commits:
+
[source]
----
feat: add support for streaming responses
fix: reduce WASM binary size by 20%
docs: add integration example for Svalinn
proof: simplify noLostRequests theorem
----
7. Submit PR with clear description

== Community

* **Questions**: Open a GitHub Discussion
* **Bugs**: Open an issue with `bug` label
* **Features**: Open an issue with `enhancement` label
* **Proofs**: Open an issue with `formal-verification` label
